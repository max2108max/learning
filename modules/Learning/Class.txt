Class нужен для создания однотипных обьектов.
В классы можно записать функции которые будут храниться в протатипах.
Мы решаем проблемы с помощью Class
Не повторяемся
Экономим память

Class это сахар над конструкторами, протатипами и дескприпторами.

статические ствойства принадлежат классу в целом, а не отдельным экземплярам этого класса

нельзя стрелку в статических методах!

у инстансов нету статических методов.

стрелка внутри класс указывает на this внутри класса

// static        this = Class
// non static    this = instance
